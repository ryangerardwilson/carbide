# __PROJECT_NAME__ Agent Context

This app was generated by Carbide.

## Runtime Shape

The local stack has three Docker Compose services:

- `web`: Bun, React, Tailwind, browser routes, and the same-origin API proxy.
- `api`: Go HTTP/API server, auth, sessions, validation, and JSON responses.
- `db`: Postgres durable state for users and sessions.

Every root directory except `agents.d/` maps to a standalone Docker service:
`web/`, `api/`, and `db/`. Shared runtime coordination lives in
`docker-compose.yml`.

## Operating Context

Read the narrow file in `agents.d/` before changing the matching area:

- `agents.d/ENVIRONMENT.md`: environment variables, local defaults, and secret
  handling.
- `agents.d/DEPLOY.md`: preview-before-apply deployment rules.
- `agents.d/BACKUP_RESTORE.md`: local reset and production backup/restore
  expectations.
- `agents.d/TAILWIND_COMPONENTS.md`: Tailwind component class organization,
  starter component ownership, and accessibility rules.
