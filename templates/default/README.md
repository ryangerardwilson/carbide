# __PROJECT_NAME__

This app was generated by Sealion.

## Start

```sh
sealion run dev
```

Then open:

```text
the app URL printed by sealion run dev
```

If port 8080 is already in use, `sealion run dev` selects another local port.
To choose one explicitly:

```sh
SEALION_HTTP_PORT=18080 sealion run dev
```

The app listens on port 8080 inside the container. Your browser should use the
host URL printed by the CLI and app logs.

`sealion run dev` starts Docker Compose watch when your Compose version supports
it. Edits under `src/`, `view/`, or to `Dockerfile` rebuild and replace the
app container.

## View

Edit page markup in `view/*.html`. Sealion renders escaped variables with
`{{ name }}` and trusted raw slots with `{!! content !!}`.

Demo login:

```text
admin@sealion.local
password
```
